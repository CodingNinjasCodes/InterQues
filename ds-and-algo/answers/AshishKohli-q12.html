What is Dangling Pointer?



Answer:

Here's a code snippet for :What is Dangling Pointer?

Dangling Pointer is a pointer that does not point to a valid memory location.
Dangling pointers arise when an object is deleted or deallocated, without modifying the value of the pointer,
so that the pointer still points to the memory location of the deallocated memory.
Following is an example.

// EXAMPLE
int* ptr = NULL
{
int x = 10;
ptr = &x;
}
// x goes out of scope and memory allocated to x is free now.
// So ptr is a dangling pointer now.

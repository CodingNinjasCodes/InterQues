How to hide scrollbars but keep its functionality?



Answer:

Sometimes we need to hide those default scroll-bars to make our page look more attractive.
There are options where we can make our scroll-bars more attractive. But most often, we are better off it.

So to hide the scrollbars, but still be able to keep scrolling, we can use the following code:

Hide scrollbar for Chrome, Safari and Opera

    .example::-webkit-scrollbar {
        display: none;
     }

Hide scrollbar for IE, Edge and Firefox

    .example {
       -ms-overflow-style: none; /* IE and Edge */
       scrollbar-width: none; /* Firefox */
     }


Note :   overflow: hidden   will remove the complete functionality of the scrollbar and we will not be possible to scroll inside the page.

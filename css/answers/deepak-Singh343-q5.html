Q : What are CSS Frameworks?


Answer:

CSS Frameworks are the preplanned libraries which make easy and more standard compliant web page styling.
CSS frameworks offer different modules and tools:

  • reset style-sheet
  • grid especially for responsive web design
  • web typography
  • set of icons in sprites or icon fonts
  • styling for tooltips,buttons,elements of forms
  • parts of graphical user interface like acordion,tabs,slideshow or modal windows
  • equaliser to create equal height content
  • often used CSS helper classes(left,hide)

The frequently used CSS frameworks are:
  • Bootstrap
  • Foundation
  • Semantic UI
  • Gumby
  • Ulkit

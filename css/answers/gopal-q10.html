Which command is used for the selection of all the elements of a paragraph?



Answer:

The p[lang] command is used for selecting all the elements of a paragraph. 

                        For Example:
                        
                        <html>
                            <head>
                                <style>
                                    p:lang(it) { 
                                        background: yellow;
                                    }
                                </style>
                                </head>
                            <body>

                                <p>I live in Italy.</p>
                                <p lang="it">Ciao bella!</p>

                                <p><b>Note:</b> For :lang to work in IE8, a DOCTYPE must be declared.</p>

                                </body>
                                </html>

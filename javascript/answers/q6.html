What is"use strict" in javaScript?



Answer:

The "use strict" was introduced in ECMAScript version 5. It is not a statement but a literal expression. With strict mode you cannot use undeclared variables. It means that the code is executed in strict mode and it is declared in 1st line of the script or a function. Declared at the beginning of a script, it has global scope (all code in the script will execute in strict mode) while declared inside a function, it has local scope (only the code inside the function is in strict mode):

Example 1:

"use strict";
x=3.14; //This will cause an error because x is not declared

Example 2:

x=3.14; //This will not cause an error
function myFunction()
{
"use strict";
y=3.14; //This will cause an error
}

Difference between == and === operator?



Answer:

The ‘==’ operator tests for abstract equality i.e. it does the necessary type conversions before doing the equality comparison.
On the other hand, the ‘===’ operator tests for strict equality i.e it will not do the type conversion hence if the two values are not of the same type, when compared, it will return false.
In simple words, "===" is a stricter equality test and returns false if either the value or the type of the two variables are different.

Input:

document.write(9 == "9");
document.write('<br>');
document.write(9 === "9");


Output:

true
false

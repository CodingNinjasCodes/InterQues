Difference between setTimeout and setInterval?



Answer:

setTimeout is a function which is used to perform an action after a given interval once whereas setInterval is a function which is used to perform a task after a interval repetitively until the interval is cleared . The format is setTimeout(function(){action..},1000) and setInterval(function(){},1000) where the time is in milliseconds.

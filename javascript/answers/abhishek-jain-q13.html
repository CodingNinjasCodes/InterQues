What is the difference between == and ===?


Answer:

The == operator checks equality only whereas === checks equality, and data type, i.e., a value must be of the same type.

Example..Strict equality (===) means values which we are comparing must have the same type. This means "2" will not be equal to 2 ("2"===2 it will return false)
Type converting equality (==) means automatically it will covert the variable to value irrespective of data type; either it is a string or a number. This means "2" will be equal to 2 ("2" == 2 it will return true).

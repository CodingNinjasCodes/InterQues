JavaScript Questions

Below is a compiled list of JavaScript interview questions that will help you conquer that upcoming interview of yours!

  1. What will the code below output to the console and why?

    var myObject = {
      foo: "bar",
      func: function() {
       var self = this;
       console.log("outer func: this.foo = " + this.foo);
       console.log("outer func: self.foo = " + self.foo);
       (function() {
        console.log("inner func: this.foo = " + this.foo);
        console.log("inner func: self.foo = " + self.foo);
       }());
      }
    };
    myObject.func();

  2. What will be the output of the code below?

    var x = { foo : 1};
    var output = (function(){
       delete x.foo;
       return x.foo;
      })();

    console.log(output);

  3. What is the 'Strict' mode in JavaScript and how can it be enabled?

  4. What are escape characters?

    Answer:

    Escape characters (Backslash) is used when working with special characters like single quotes, double quotes, apostrophes and ampersands. Place backslash before the characters to make it display.

    document.write "I m a "good" boy"
    document.write "I m a \"good\" boy"

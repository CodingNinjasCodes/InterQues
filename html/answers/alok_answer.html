What is the difference between HTML elements and tags?



Answer: An element is a set of opening and closing tags in use. Tags are labels you use to mark up the begining and end of an element. All tags have the same format: they begin with a less-than sign "" and end with a greater-than sign ">". ... The only difference between an opening tag and a closing tag is the forward slash "/"

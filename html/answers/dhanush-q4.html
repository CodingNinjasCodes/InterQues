Explain the importance of the HTML DOCTYPE?



Answer:

  • DOCTYPE is an instruction to the web browser regarding what version of the markup language the page is written.
  • The DOCTYPE declaration need to be the pretty first thing in your HTML document, before the tag.
  • Doctype declaration points to a Document Type Definition (DTD).
  • It provides markup language rules, so a browser can interpret the content correctly.
